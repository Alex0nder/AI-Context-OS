# MA10 (A)

## Answer

The required environment variables for the Worker are:

- `DATABASE_URL`
- `RESEND_API_KEY`
- `RESEND_WEBHOOK_SECRET`
- `API_KEY`
- `INBOX_DOMAIN`
- `DEFAULT_TTL_MINUTES`
- `MAIL_QUEUE`
- `INBOX_WAIT`

Optional variables include:

- `RATE_LIMIT_KV_WRITE_EVERY`
- `RATE_LIMIT`
- `RATE_LIMIT_PER_MINUTE`
- `STRIPE_SECRET_KEY`
- `STRIPE_WEBHOOK_SECRET`
- `STRIPE_PRICE_PRO`
- `MCP_OAUTH_TOKEN_TTL_SEC`
- `MCP_OAUTH_JWT_SECRET`
- `OIDC_ISSUER`
- `OIDC_CLIENT_ID`
- `OIDC_CLIENT_SECRET`
- `OIDC_AUDIENCE`
- `RAW_MIME`
- `RAW_MIME_MAX_BYTES`
- `RAW_MIME_AGENT_MAX_BYTES`
- `ATTACHMENT_MAX_STORE_BYTES`
- `OUTBOUND_FROM`
- `AI`
- `SEARCH_EMBED_MODEL`
- `EXTRACT_MODEL`
- `AUDIT_RETENTION_DAYS`
