# MA29 (B)

## Answer

To use Playwright with MailAgent, follow these steps:

1. **Install Playwright**:
   - Ensure you have Playwright installed in your project.

2. **Create a Test**:
   - Use the Playwright framework to create a test script that interacts with the MailAgent API.

3. **Example Directory**:
   - Refer to the `examples/playwright/` directory for sample tests.

4. **Run Playwright Tests**:
   - Execute your Playwright tests using the command:
     ```bash
     npm run test:pw:simulate
     ```

5. **Documentation**:
   - For detailed guidance, refer to **docs/AUTOTESTS.md** and **AGENTS.md** for the testing matrix and what to run after changing specific components.

Make sure to set up your environment variables as needed for the tests to run successfully.
