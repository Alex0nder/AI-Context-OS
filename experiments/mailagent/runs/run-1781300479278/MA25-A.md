# MA25 (A)

## Answer

To create a team API key, you can use the following command:

```bash
npm run issue:key:db -- <team-name>
```

Replace `<team-name>` with the desired name for your team. This command will generate a new team API key and display it along with the `team_id`. 

Make sure to save the output, as the key will be shown only once.
