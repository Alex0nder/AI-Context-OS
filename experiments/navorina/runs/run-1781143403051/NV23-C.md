# NV23 (C)

## Answer

Navorina has the following retention levers:

- **Retention Nudges**: 
  - Various types of nudges based on user activity and onboarding status, such as:
    - `needs_snapshot`
    - `brief_after_snapshot`
    - `verdict_after_snapshot`
    - `monthly_close`
    - `setup_almost_done`
    - `weekly_brief`

- **Retention Nudge Actions**: 
  - Actions associated with nudges, including:
    - `snapshot`
    - `brief`
    - `insights`
    - `budget`
    - `income`
    - `expense`
    - `verdict`

- **Dismissal Mechanism**: 
  - Users can dismiss nudges, which are tracked using local storage.

- **Pending Briefs**: 
  - Mechanism to mark and check for pending briefs after snapshots.

- **Snapshot Streak Tracking**: 
  - Tracks consecutive months with saved snapshots to encourage regular usage.

- **Weekly Briefing**: 
  - Provides a summary of weekly activities and insights to keep users engaged.

- **AI Verdict Suggestions**: 
  - Auto-suggests AI-generated verdicts after saving a monthly snapshot to enhance user interaction.

These levers are designed to encourage user engagement and retention by providing timely reminders and insights based on user behavior.
