# NV11 (A)

## Answer

The first value for new users is blocked by:

- **Fragmented entry points**: Different experiences across landing, assistant, tools, and Nucleus.
- **Lack of contextual AI**: AI is separate from Financial Mirror workflows, leading to disjointed user experiences.
- **Static URLs**: Many tools have static URLs without integration with Financial Mirror, making navigation less intuitive.
- **Voice retention**: The voice feature is not effectively integrated to enhance user engagement.

These issues hinder a seamless onboarding experience and user engagement with the platform.
