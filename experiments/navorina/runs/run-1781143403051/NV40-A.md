# NV40 (A)

## Answer

The process for accepting crypto on invoices in the Navorina project involves the following steps:

- **Payment Link**: A payment link is generated for the invoice that allows clients to pay using cryptocurrency.
- **Webhook Integration**: The system uses webhooks to handle payment statuses, such as paid, underpaid, or overpaid.
- **User Interface**: The invoice will display options for accepting crypto payments, which are integrated into the invoicing system.
- **Compliance**: There are compliance measures in place for crypto transactions, which are detailed in the relevant documentation.
- **Testing and Validation**: The functionality is tested through various scripts to ensure it works correctly before going live.

For more detailed information, refer to the specific sections in the documentation regarding crypto payments and invoicing.
