# NV34 (A)

## Answer

When editing Nucleus UI tabs, the following rules apply:

- **Use Tokens**: Ensure that styles are based on tokens from `Navorina/css/variables.css` instead of using raw pixel values.
- **Shared Layout**: Copy shared blocks (header, footer, navigation) exactly as per the established layout without redesigning.
- **Component Consistency**: Follow the guidelines in `docs/design-system/COMPONENT_LIBRARY.md` for forms, including height, select options, and date icons.
- **Mobile Considerations**: Ensure that the mobile shell is accounted for in app pages, adhering to the mobile design principles.
- **Modern Web Guidance**: Apply Modern Web Guidance for any dialog, popover, or tab interactions if relevant.
- **Crypto Terminology**: Verify that the terminology used aligns with the guidelines in `docs/planning/CP_CRYPTO_PAYMENTS_TERMINOLOGY.md`.
- **Scope Agreement**: Ensure that the changes align with the product scope as defined in the PRD and roadmap.
