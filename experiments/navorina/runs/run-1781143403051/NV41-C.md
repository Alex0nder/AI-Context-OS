# NV41 (C)

## Answer

The CP8 crypto compliance in Settings refers to the following aspects:

- **Compliance Links**: It includes links to external compliance policies and disclaimers related to crypto payments.
- **Acknowledgment**: Users must acknowledge compliance with crypto payment regulations, which is stored in local storage.
- **Disclaimers**: It provides disclaimers regarding crypto payments, including information about tracking, irreversibility, and AML/KYC responsibilities.
- **Provider Awareness**: The compliance links are provider-aware, meaning they can change based on the payment provider being used.
- **User Interface**: The compliance settings are mounted in the user interface, allowing users to view and interact with the compliance information.

These elements ensure that users are informed about the compliance requirements when accepting crypto payments.
