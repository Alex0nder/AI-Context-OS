# NV34 (B)

## Answer

When editing Nucleus UI tabs, the following rules apply:

- **Shared Header/Footer**: Follow the exact copy rule for the shared header and footer.
- **UX Principles**: Refer to `UX_PRINCIPLES.md` for guidelines on user experience.
- **Design Tokens**: Use `variables.css` for design tokens.
- **Component Map**: Consult `COMPONENT_MAP.md` for component usage and structure.
- **Pre-Edit Search**: Use `npm run mwg:search` to check for existing implementations and avoid duplication.
