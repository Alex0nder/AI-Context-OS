# NV08 (A)

## Answer

The modules in the Navorina PRD scope include:

- **Marketing and Content**
  - Landing page (`index.html`)
  - Documentation and contacts (`docs.html`, `contact.html`)
  - Localizations (`ru/`, `es/`)

- **Assistant (Core Experience)**
  - Assistant page with chat and 3D sphere (`assistant.html`)
  - Components: `Chat`, `ChatInput`, `ChatMessage`, `Sphere`, `InstrumentWidget`
  - Voice integration

- **App Suite / Embedded Content**
  - Command palette and content rendering (`appsuite.html`)

- **Nucleus — Financial Mirror**
  - Hub and separate screens for dashboard, budget, income, transactions, taxes
  - Logic for income, expenses, snapshots, currencies, and guest workspace

- **Tools (Static Generators)**
  - Various generators (invoice, QR, document, proposal, quote)

- **Account Management**
  - Registration, login, password reset pages (`auth.html`, `signup.html`, `reset-password.html`)

These modules collectively support the core functionalities of the Navorina product as outlined in the PRD.
