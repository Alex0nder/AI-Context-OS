# NV39 (B)

## Answer

To check workspace services without live keys, you can use the following steps:

- Run the integration pilot verification using the command: `npm run verify:integrations-pilot`.
- Ensure that you are using the emulators specified in `server/development.defaults.env`, which allows testing without live keys.
- Refer to the documentation for integration checks in `docs/planning/INTEGRATIONS_WHEN_READY.md`.
