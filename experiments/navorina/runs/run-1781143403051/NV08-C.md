# NV08 (C)

## Answer

The modules in the Navorina PRD scope include:

- **Assistant**: 
  - Chat interface and 3D sphere
  - Components: Chat, ChatInput, ChatMessage, Sphere, optional InstrumentWidget
  - Voice integration with STT/TTS utilities

- **Nucleus (Financial Mirror)**:
  - Hub for financial management
  - Screens: nucleus-dashboard, nucleus-budget, nucleus-income, nucleus-transactions, nucleus-taxes
  - Logic for income, expenses, snapshots, currencies, and guest workspace

- **App Suite**:
  - Command palette and content rendering

- **Tools**:
  - Static generators for invoices, QR codes, documents, proposals, quotes, and calculators

- **Account Management**:
  - Registration, login, password reset functionalities

These modules are part of the MVP (Minimum Viable Product) for the Navorina project.
