# OL06 (C)

## Answer

When a user requests "organize downloads," the following occurs:

- The request is processed by the Oiloop assistant.
- If the user has set a rule to organize downloads, Oiloop will suggest organizing the files in the Downloads folder.
- The assistant will check the user's workspace configuration and the number of loose files in the Downloads folder.
- If the workspace is configured and there are loose files, Oiloop will provide a notification or message indicating that it can help organize those files.
- The action will only proceed with explicit user confirmation, as Oiloop does not move or delete files without user consent.
