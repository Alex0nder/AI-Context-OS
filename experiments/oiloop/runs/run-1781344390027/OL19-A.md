# OL19 (A)

## Answer

Oiloop groups browser tabs into new windows using the following process:

- **Identify Groups**: It identifies tabs that belong to the same domain.
- **Move Tabs**: The service moves these grouped tabs into new windows based on their domain.
- **Confirmation**: Users can confirm the action to execute the grouping of tabs.

This functionality is part of the `SafariService` class, which handles the preparation and confirmation of tab grouping actions.
