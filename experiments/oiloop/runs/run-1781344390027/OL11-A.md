# OL11 (A)

## Answer

The Telegram and Slack connector drafts are structured as follows:

### Telegram Draft Structure
- **Draft ID**: Unique identifier for the draft.
- **Bot Token**: Token used for authentication with the Telegram Bot API.
- **Chat ID**: Identifier for the chat where the message will be sent.
- **Message Content**: The body of the message to be sent.

### Slack Draft Structure
- **Draft ID**: Unique identifier for the draft.
- **Token**: User or bot token for authentication with the Slack API.
- **Channel**: The target channel where the message will be sent.
- **Message Content**: The body of the message to be sent.

Both drafts include necessary identifiers and content to facilitate sending messages through their respective platforms.
