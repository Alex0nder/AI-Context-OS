# OL06 (C)

## Answer

When a user requests "organize downloads," the following occurs:

- The request is processed by the Oiloop assistant.
- If the user has set a rule to organize downloads, the assistant will acknowledge this and proceed to organize the specified folder.
- The assistant will check the user's workspace configuration to ensure it is set up correctly.
- If there are loose files in the Downloads folder, the assistant will help organize them based on the user's preferences.
- The action will only be taken with explicit user confirmation, as Oiloop does not move or delete files without user consent.
