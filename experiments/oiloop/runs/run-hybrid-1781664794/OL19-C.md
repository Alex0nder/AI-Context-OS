# OL19 (C)

## Answer

Oiloop groups browser tabs into new windows using the following process:

- **Tab Grouping Logic**: The `SafariService` manages the logic for regrouping open tabs by domain or category.
- **AppleScript Execution**: It executes AppleScript commands to create new browser windows and move the grouped tabs into them.
- **New Window Creation**: A new window is created for each group of tabs, and the tabs are added to this new window.
- **Automation Permissions**: This process requires Safari Automation permissions to function correctly.

This functionality is part of the browsing core's scope, specifically designed for both Safari and Google Chrome.
