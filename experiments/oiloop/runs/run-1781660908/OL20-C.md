# OL20 (C)

## Answer

The settings controls for browser integration in Oiloop include:

- **Browser Automation Settings**: Users can enable or disable browser automation for Safari and Chrome through the application settings.
- **Safari Settings**: Specific settings for Safari can be managed, including enabling or disabling the fetching and closing of tabs.
- **Chrome Settings**: Similar to Safari, settings for Chrome can be configured to allow fetching and closing of tabs.
- **User Confirmation**: Any actions related to browser tab management require user confirmation, ensuring no silent operations occur without explicit user consent. 

These settings are accessible under **Settings → Routine automation**.
