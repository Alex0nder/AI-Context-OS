# CANON — example-saas

**Context OS** stores **decision domains**, not file dumps.

- **Cores** answer: *what do I need to decide X?*
- **Router** picks 1–2 cores before the LLM runs
- **Sources** in each core are pointers — open selectively

Default pipeline for this scaffold: **B** (routed cores). Full-repo context is baseline only.

Research: https://github.com/Alex0nder/AI-Context-OS
