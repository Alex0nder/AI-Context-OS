# Product Specification

The primary user is a product or engineering lead who needs trustworthy weekly
metrics without maintaining an analytics pipeline. The first-run journey offers
sample data, then asks the owner to connect a supported source, test the
connection, synchronize events, and save a dashboard.

Sample mode requires no external credentials. Production activation requires a
successful synchronization. Connection tests return actionable errors and
preserve entered values after failure.

The current roadmap order is connection diagnostics, dashboard sharing, then
additional chart types. Success is measured by time to first dashboard,
successful source connection rate, and seven-day dashboard return rate.
